数据库

第一章 绪论

1.1 数据库系统概述

1.1.1 数据库的四个基本概念

数据
  • 数据是描述事物的符号。数据的含义称为数据的语义，数据与语义不能分开。
数据库(DB:database)
  • 数据库：存放数据的仓库
  • 数据库是长期存储在计算机内，有组织的，可共享的大数据的集合。
  • 数据库的特点：
    • 具有一定的模型组织、描述，存储。
    • 具有较小的冗余度。
    • 较高的数据独立性。
    • 具有易扩展性。
    • 具有较高的用户共享性。
  • 数据库的特点概括：
    1. 永久存储。
    2. 有组织。
    3. 可共享。

第一章 Java 8 新特性

  • Java 8 新特性主要包括：
    • Stream API
    • 向方法传递代码的技巧
    • 接口中的默认方法

1.1 流处理

  • 流是一系列的数据项，一次只生成一项。
  • 程序可以从输入流中一个一个读取数据，以同样的方式将数据写入输出流。
  • 一个程序的输入流可能是另一个程序的输出流。
  • Java 8 借助流水线的思想，在java.util.Stream中添加一个Stream API，Stream<T>就是一系列T类型的项目。
  • Java 8 可以透明地把输入的不相关的部分拿到几个CPU内核上面去分别执行的操作流。
